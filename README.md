# AI-Native Book Project

This is an AI-Native Physical & Humanoid Robotics Textbook with personalization and translation capabilities.

## Features

- Docusaurus-based documentation site
- Content personalization based on user preferences
- Multi-language support (including Urdu translation)
- Interactive textbook components

## Installation

### Prerequisites

- Node.js 18+
- npm or yarn package manager

### Setup

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run start
```

3. The application will be available at `http://localhost:3000`

## Project Structure

- `docs/` - Textbook content and chapters
- `src/` - Custom React components and styling
- `i18n/` - Internationalization files
- `blog/` - Blog posts related to the textbook

## Deployment

This project is configured for deployment with Vercel.

1. Push your code to the GitHub repository
2. Connect your repository to Vercel
3. Vercel will automatically build and deploy your application
